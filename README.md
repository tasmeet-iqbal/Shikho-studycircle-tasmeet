# StudyCircle

A working v1 prototype of StudyCircle, a class-based social and study layer for the Shikho app (Bangladesh K-12 edtech, classes 9 to 12).

It goes after activation: getting a newly registered student to their first real lesson, fast, with a cooperative class framing. The home opens on a "your first class is ready" card that takes you straight into the free lesson in one tap, value before any payment. Progress is saved in the browser, so it persists across a reload.

## Run it
Open `index.html` in a browser, or visit the deployed link.

## In this repo
- `index.html` plus `fonts/` and `assets/`: the working clickable prototype.
- `studycircle-plan.md`: the v1, v2, v3 plan and the calls made.
- `build-trace.md`: how it was built and the key decisions.

Built in plain single-file HTML and CSS, grounded in the real Shikho UI. Cooperative, never a leaderboard. Anonymous, since the users are minors. Honest counts. SVG icons, no emoji.
